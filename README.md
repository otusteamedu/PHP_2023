#### результат выполнения домашней работы 03

##### Установка

Запуск команды установки composer пакетов
```bash
composer install
```

##### Настройка

Создание файла настроек на основе примера
```bash
cp .env.example .env
```

##### Запуск

Пример запуска php-файла
```bash
php -f src/index.php
```

###### Ожидаемый вывод
Такой вывод ожидается, если все корректно работает
```bash
VAL-KEY
+
-not-exist-
```
