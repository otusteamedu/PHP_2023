Доступные команды

1. Создание бд (индекса) (Ответ Json от эластика)

   ``` php app.php create_db```


2. Удаление бд (удаление индекса) (Ответ Json от эластика)

   ``` php app.php delete_db```


3. Импорт данных в бд (bulk) (сделал более простой ответ)

      ``` php app.php import_db```


4. Поиск данных по ТЗ (параметры захардкодил, так как по ТЗ этого не нужно)
   Ответ преобразовал в <b>таблицу<b>

   ``` php app.php search_books```
