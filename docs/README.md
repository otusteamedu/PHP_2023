### Инициализация:

Для запуска необходима утилита [mkcert](https://github.com/FiloSottile/mkcert)
```bash
make init
make index
```
### Запуск
```bash
make up
```
### Остановка
```bash
make down
```

### Проверка работы elasticsearch
```bash
make check
```

## Создание индекса и заполнение индекса данными
Из корня репозитория выполнить команду:
```bash
make index
```
